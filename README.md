# Network-Science
- Objective: Represent a dataset as an organizational network and investigate the various nodes comprising the network.
- Dataset: Organizational data taken from Kaggle site that is made up of more than 10,000 entries, showcasing which organization 
- Approach: Load in raw dataset and use centrality calculations to determine influential nodes.
- Key Findings: The most influential organizations in this network were Microsoft and Azure, while the most influential developers appeared to be Lucas Santos and Darcy Clarke.
